# Manual básico de usuario

## Acceso
- URL: `http://10.0.0.3:18080`
- Usuario inicial: `admin@ionlife.local`
- Contraseña: `Admin123!`

## Módulos
1. Clientes: registro y consulta de clientes.
2. Productos: catálogo y precios.
3. Almacenes: gestión de ubicaciones e inventario.
4. Pedidos: creación y seguimiento de estados.
5. Logística: camiones, repartidores y asignación.
6. Reportes: ventas, pedidos por estado, stock y rendimiento.

## Flujo recomendado
- Crear cliente y dirección.
- Registrar productos y stock.
- Crear pedido desde call center.
- Confirmar pedido y registrar salida de inventario.
- Asignar camión y repartidor.
- Confirmar entrega.
